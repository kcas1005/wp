Document Object Model로 HTML, XML 문서에 접근하기 위한 인터페이스입니다. 문서 내의 모든 문서를 정의하고, 각각의 요소에 접근하는 방법을 제공합니다. DOM의 종류 Core DOM : 모든 문서 타입을 위한 DOM 모델 HTML DOM : HTML 문서 타입을 위한 DOM 모델 XML DOM : XML 문서 타입을 위한 DOM 모델 2.CSS display 속성의 속성값을 분류하며 설명해 주세요 - 유빈 3.CSS content, padding, border, margin과 position에 대해 설명해주세요 - 진실 content : 박스에 들어가는 텍스트나 이미지 등의 내용물 padding : 콘텐츠와 경계선 사이의 간격 border : 콘텐츠와 패딩을 포함한 경계를 나타내는 선 margin : 경계선과 외부 요소 사이의 간격 position : 4.git에 대해 설명해주세요 (github 포함) - 나 Git 이란? 소스코드의 최신판을 관리하기 어렵기 때문에 생긴 형상 관리 도구 (Configuration Management Tool)입니다. Git 특징 여러명이 동시에 작업할 수 있는 병렬 개발이 가능합니다. 프로그램의 변동 과정을 체계적으로 관리할 수 있고 지난 시점의 소스코드로 점프 할 수 있습니다. 새로운 기능을 추가하는 Experimental version을 개발하는 경우, 실험을 충분히 한 후 합치는 방식인 Merge로 개발을 진행 할 수 있습니다. 분산 버전관리라서 인터넷이 연결되지 않는 곳에서 개발을 할 수 있고 중앙 저장소가 폭파되어도 다시 원상 복구 할 수 있습니다.
